HTML elements consist of start tag, content and end tag.

Start Tag Content End Tag
< p > Content Goes here < /p >

List of Html Elements and Group by Function

Main Root

  • < html >
    represents the root element and it defines the whole document.

Document Metadata

Information about the page such styles, script and data.

  • < base >
    Specify the base URL and it can be only one base element in a document
  • < head >
    Contains information such as title, styles , and scripts.
  • < link >
    Specify the relation between a html document to other resources.

Content Sectioning

Used in organizing document content into logical pieces.

  • < address >
    Represent the contact information for its nearest article or body elements ancestors. Contact information of the owner or author of the site.
  • < article >
    Content of a page that can be reusable and it is possible to use it independently to the rest if the site. Example of this are the forum post, blog post, news story, and comments.
  • < aside >
    Content aside from the page content.
  • < footer >
    Usually contains information such as copyright , links to the related documents and author of the section.
  • < h1 - h6 >
    Section heading of the site. Heading are numbered by the decreasing importance.

    H1 Sample

    H2 Sample

    H3 Sample

    H4 Sample

    H5 Sample
    H6 Sample
  • < header >
    Represents a group introductory or navigational aids.
  • < nav >
    A section with a navigation links.
  • < section >
    Defines the section such as chapter, header, footers, and others.
