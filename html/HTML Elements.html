HTML Elements

Next

HTML elements consist of start tag, content and end tag.

Start Tag Content End Tag
p Content Goes here /p

List of Html Elements and Group by Function

Main Root

html
Represents the root element and it defines the whole document.

Document Metadata

Information about the page such styles, script and data.

base
Specify the base URL and it can be only one base element in a document.
head
Contains information such as title, styles , and scripts.
link
Specify the relation between a html document to other resources.

Content Sectioning

Used in organizing document content into logical pieces.

address
Represent the contact information for its nearest article or body elements ancestors. Contact information of the owner or author of the site.
article
Content of a page that can be reusable and it is possible to use it independently to the rest if the site. Example of this are the forum post, blog post, news story, and comments.
aside
Content aside from the page content.
footer
Usually contains information such as copyright , links to the related documents and author of the section.
h1 - h6
Section heading of the site. Heading are numbered by the decreasing importance.
header
Represents a group introductory or navigational aids.
nav
A section with a navigation links.
section
Defines the section such as chapter, header, footers, and others.

Text content

dd
Represent the description list.
dt
Represents the descrption.
ficaption
Represent a caption for a figure element.
figure
Self contained content usually with. figcaption
hr
Thematic break.
li
Represent an item in a list.
main
Main content of the body.
ol
ordered list f items usually rendered as a number lists.
p
Text paragraph.
pre
Preformatted text.
ul
Undoredered list usually rendered as bulleted list.

Inline Text Semantics

a
Anchor element that create a hyperlink to another document.
abbr
Abbreviation
b
Use to bold a text or words.
bdi
Bidirectional isolation. Isolates a part of text that might be formatted in a different direction from other text outside it.
bdo
Bidirectional override Text direction override.
br
line break (carriage-return).
cite
Creative work references. Defines the title of the work.
code
Piece of computer code.
dfn
Define instance of term.
em
Emphasis
i
Italicize words ot text.
kbd
User input or keyboard input.
mark
Highlighted text.
q
Short quotes.
rp
Defines what to show in browsers that do not support ruby annotations
rt
Defines an explanation/pronunciation of characters (for East Asian typography)
ruby
Represent Ruby Annotation
s
Strikethrough
samp
Sample output of a program.
small
Smaller text.
span
Section in Document
strong
Defines important text
sub
Subscripted text
sup
Superscripted text
time
Defines a date/time
u
Defines text that should be stylistically different from normal text. Underlin a text or words
var
Represents a variable in a mathematical expression or a programming context.
wbr
Defines a possible line-break.

Image and Multimedia

area
Defines an area inside an image-map
audio
Defines sound content
img
Image
map
Client-side image-map
track
Defines text tracks for media elements (video and audio)
video
Defines a video or movie

Embedded content

embed
A container for an external (non-HTML) application
object
Embedded object.Integration point for an external application or interactive content.
param
Parameters for an object element.
source
Defines multiple media resources for media elements (video and audio).

Scripting

canvas
Used to draw graphics, on the fly, via scripting (usually JavaScript)
noscript
An alternate content for users that do not support client-side scripts
script
Client-side script

Demarcating Edits

del
Defines text that has been deleted from a document
ins
A text that has been inserted into a document

Table Content

caption
A table caption
col
Specifies column properties for each column within a colgroup element
colgroup
Specifies a group of one or more columns in a table for formatting.
table
Table.
tbody
Groups the body content in a table.
td
A cell in a table.
tfoot
Groups the footer content in a table.
th
A header cell in a table.
thead
Groups the header content in a table.
tr
A row in a table.

Forms

button
A clickable button.
datalist
Specifies a list of pre-defined options for input controls.
fieldset
Groups related elements in a form.
form
An HTML form for user input.
input
An input control.
label
A label for an input element.
legend
A a caption for a fieldset element.
meter
A scalar measurement within a known range (a gauge).
optgroup
Creates a grouping of options within a select element.
option
An option in a drop-down list.
output
The result of a calculation.
progress
Represents the progress of a task.
select
A drop-down list.
textarea
A multiline input control (text area).
