HTML elements consist of start tag, content and end tag.

Start Tag Content End Tag
< p > Content Goes here < /p >

Empty elements are html element with no content and it doesn't have end tag. Example < br >, br element indicates a line break

List of Html Elements and Group by Function

Main Root

  • < html >
    represents the root element and it defines the whole document.

Document Metadata

Information about the page such styles, script and data.

  • < base >
    Specify the base URL and it can be only one base element in a document
  • < head >
    Contains information such as title, styles , and scripts.
  • < link >
    Specify the relation between a html document to other resources.
