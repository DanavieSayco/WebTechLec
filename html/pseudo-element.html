CSS Pseudo-elements

Previous

Next

Pseudo-elements enables an author to refer to a content that does not exist in the document. It can also enable the author to refer to the first letter or first line of an elementâ€™s content. Only one pseudo-element can be used in a selector.

The double colon notation (::) was introduced in CSS3 to help differentiate pseudo-classes from pseudo-elements. In previous versions of CSS, a single colon notation was used for both pseudo-classes and pseudo-elements. For compatibility sake, single colon notation is still accepted but future pseudo-elements should follow the double colon notation.

::first-letter

The ::first-letter pseudo-element refers to the first letter of an element. It must not be preceded by other content to be selected. Punctuation after the first letter is included as well. If the first letter is a digit, the pseudo-element still works. This pseudo-element only applies to block-level elements.

p::first-letter {
    color: coral;
    font-size: 30px;
}

This is a paragraph.

::first-line

The ::first-line pseudo-element selects the first line of a text. Just like the previous pseudo-element, it only applies to block-level elements.

p::first-line {
    color: orange;
}

This is a paragraph. A paragraph this is. Paragraphs are made up of words. Words are made up of letters. Letters are made up of lines and curves. Lines and curves are made up of points. ::before

The ::before pseudo-element is used to insert a content before an element.

p::before {
    content: url('pictures/tardis.gif');
}

This is a paragraph with an image before it.

::after

The ::after pseudo-element is used to insert a content after an element.

p::after {
    content: url('pictures/tardis.gif');
}

This is a paragraph with an image after it.
