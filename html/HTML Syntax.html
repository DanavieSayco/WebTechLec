Sample HTML Document

<!DOCTYPE html>
Rendering a web page using standards will make the presentation of the web page consistent throughout different browsers.When not indicated in the HTML document, the web page will be rendered using quirks and the presentation might differ from one browser to another.
<head></head>
Contains metadata such as links, information of author, and others.
<body></body>
Contains paragraph and content.

Anatomy of Element

Opening tag
Consist of angle brackets and name of elements
Closing tag
Consist of angle brackets, a forward slash and the name of elements. This is where the element ends . It can lead to strange result if failed to include closing tag and this is common beginner error.
Content
Consist of text.
Element
Consist of opening tag, content and closing tag.

Nesting Elements

Putting element inside an element.

Example:

<p> This is <strong> important </strong> </p>

Output: This is important

Difference of Block and Inline Elements

Block element - Block element will appear next line
Example: <p> Sample<xmp><p> Block Element</p>

Output:Sample

Block Element

Inline Element - Inline element will not cause new line to appear
Example: <p> Sample <strong> Inline Element </strong></p>
Output:Sample Inline Element

Empty Elements

Empty elements are html element with no content and it doesn't have end tag. Example < br >, br element indicates a line break

Attributes

Attribute is added in the start tag and act as modifier of html element. Contains other information of elements.

HTML Terminology

Element
Contains Start Tag, Content and End Tag
Tag
Start tag contain angle bar . The end bar contains front slash and angle bar.
Attribute
Contain an equal sign, the name of attribute and the attribute value.
Browser
Used for accessing and displaying HTML
Comments
Information for future references.
Broken Links
Links that don't work
Deprecated Element
The element can be removed anytime in the future.
Link
Used to connect one document to another document
Syntax
Rules of a computer language to perform tasks.
