JSP Actions

Previous

Next

JSP actions are special XML tags which control the behavior of servlet engine. They allow you to insert a file dynamically, reuse external JavaBean components, forward the request to the other page and generate HTML for Java Applet plugin.

Syntax:

Action Elements

  • jsp:include

    • Includes the response from the jsp page while processing.

  • jsp:useBean

    • Loads a JavaBean component into the page and use it later. jsp useBean action allows you to reuse other Java classes.

  • jsp:setProperty

    • Sets the property of a JavaBean.

  • jsp:getProperty

    • Get a property value from JavaBeans component and adds to the response

  • jsp:forward

    • Forwards the request to the other page.

  • jsp:plugin

    • Generates browser-specific code that makes an OBJECT or EMBED tag for the Java plugin.

  • jsp:param

    • Passes the parameters to other JSP action tags such as JSP include and JSP forward tag.

  • jsp:element

    • Dynamically defines XML elements.

  • jsp:attribute

    • Sets the value of the action element based on the body of this element.

  • jsp:body

    • Sets action element body based on the body of this element.

  • jsp:text

    • Used to write template text in JSP pages and documents.

  • jsp:fallback

    • Used in jsp:plugin. It can be used to print the message if the plugin is working
