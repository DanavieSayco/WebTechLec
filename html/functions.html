JavaScript Functions

Previous

Next

A JavaScript function is a block of code designed to perform a particular task. It is executed when invoked or called upon. It uses lexicographical order, and so, false is less than true. A function can be recursive, which means that a function can be called inside another function.

Syntax

A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().

function name(parameter 1, parameter 2) {
    codes are placed here;
}

It can also use the arrow syntax if used as a value to a variable.

var multiply = (a, b) => {body}

In this syntax, if it only contains one statement and it is a return statement, the braces and the return keyword can be dropped. If there is no argument, parentheses must be placed.

Invocation

Invoking as a Function

There is always a default global object in JavaScript. In HTML, the default global object is the HTML page itself. When a function is called without an owner object, the value of this becomes the global object. In a web browser the global object is the browser window.

function myFunction(a, b) {
    return a + b;
}
myFunction(5, 7);

Invoking as a Method

To invoke a function as a method, the function must belong to an object. In this type of invocation, the value of this is the object which the function belongs to.

var person = {
    firstName:"Juan",
    lastName: "Dela Cruz",
    fullName: function () {
        return this.firstName + " " + this.lastName;
    }
}
person.fullName();

Invoking with a Function Constructor

If the new keyword is used in the invocation of a function, then it is a constructor invocation. The value of this in this invocation is null.

function person(arg1, arg2) {
    this.firstName = arg1;
    this.lastName  = arg2;
}
var a = new person("Juan", "Dela Cruz");
a.firstName;
