CSS Selectors

Previous

Next

Selectors are like indicators or patterns that are used to determine which elements are to be formatted. They target specific HTML elements and apply styles to those elements. They are separated by combinators which can be whitespaces, “>”, and “+”. The elements that match a selector are called subjects.

Syntax

Simple selectors can be a type selector or a universal selector followed immediately by attribute selectors, ID selectors, or pseudo-classes. A selector with one simple selector matches any element that will satisfy its requirements. Chain of sequences of simple selectors are separated by combinators, with one pseudo-element possibly appended to the last sequence.

For the sequence of simple selectors, always start with a type selector or a universal selector. It cannot contain other type selectors or universal selectors in one sequence.

Universal selector

A universal selector represents all the elements on a page. It is represented by an asterisk (*). The properties inside the curly braces will be applied to all the elements on the HTML page.

Attribute Selectors

Attribute selectors targets elements based on their attributes.

Syntax:

  • [attr]

    • ]-Selects all elements with a specific attribute.

  • [attr=value]

    • Selects an element with an attribute with a value. The attribute must be exact.

  • [attr~=value]

    • Selects an element with an attribute whose value is separated with whitespaces. If the value contains white space, it will never represent anything.

  • [attr|=value]

    • Used for targeting the "lang" attribute.

  • [attr^=value]

    • (i.e. a[href^=”http”) Selects all < a > elements with an href attribute whose value starts with “http”.

  • [attr$=value]

    • (i.e. a[href$=”.html”) Selects all < a > elements with an href attribute whose value ends with “.html”.

  • [attr*=value]

    • Selects all elements with an attribute whose value contains whatever substring value has. If value has an empty string then it will not represent anything.
