Http Status Codes

Next

Informational (1xx)

The value of 1xx is Informational, where the request process will continue.

  • 100 – Continue

    Continuing the request of the client and the request is not yet completed or rejected by the server

  • 101 – Switching protocol

    When the client request a switch to different protocol using upgrade header

Success (2xx)

The value of 2xx is Success. It means the request is successful and accepted

  • 200 – OK

    Request is successful and the server will a response to the client

  • 201 – Created

    New resources has been created.

  • 202 – Accepted

    Request is accepted but not yet generated

  • 203 – Non-Authoritative Information

    Some are not from the origin server. Something has been added while going through a proxy

  • 204 – No Content

    Successful but not returning anything.

  • 205 – Reset Content

    client can reset forms and data entries

  • 206 – Partial Content

    In other words, this is not the whole resource.

Redirection (3xx)

The value od 3xx is Redirection. There will be an additional steps before completing the process

  • 300 - Multiple Choices

    When resource is available in different forms

  • 301 – Moved Permanently

    Resource is no longer here and location must be changed. Example pdf, java, html, text file, etc..

  • 302 – Found

    When a new resource is generated and can be found somewhere

  • 303 – See Other

    Same as 302. can be found somewhere.

  • 304 – Not Modified

    Cache and server resource are the same

  • 305 – Use Proxy

    Access through proxy. Not directly accessing origin, it is either for validation, filtering or authorization.

  • 306 – n/a (unused status code)

    Used by earlier version.

  • 307 – Temporary Redirect

    Like 301 but location must not be changed. Possibility in using when the site is in maintenance or crashed.

Client Error - 4xx

The value of 4xx is Client error, where it means that the request is not accepted and understood because the request contains syntax error.

  • 400 Bad Request

    The server did not recognized the request, the request is malformed

  • 401 Unauthorized

    The request page is a protected resource and needs clinet authentication

  • 402 Payment Required

    For digital payment, but according to Wikipedia, this code is reserved for future use.

  • 403 Forbidden

    The requested page must not be accessed, access is not allowed.

  • 404 Not Found

    The requested resource is not found in the server.

  • 405 Method Not Allowed

    The specified request method is not allowed. The request method is not supported by the resource.

  • 406 Not Acceptable

    The requested resource does not have a current representation that would be acceptable to the user agent and the server is unwilling to supply a default representation, as described in RFC 7231 Section 6.5.6

  • 407 Proxy Authentication Required

    The client must first authenticate itself with the proxy server.

  • 408 Request Timeout

    The request is taking too long to process the request.

  • 409 Conflict

    The request cannot be done due to a conflict with the current state of the requested resource.

  • 410 Gone

    The requested resource has been purposely removed and no longer available.

  • 411 Content Length Required

    "Content-length" is not specified and is required.

  • 412 Precondition Failed

    Specified precondition in the request is rejected by the server.

  • 413 Request Entity Too Large

    The specified payload it too big and it will not be accepted by the server.

  • 414 Request URI Too Long

    The server refuses to accept the request due to the request-target is too long.

  • 415 Unsupported Media Type

    the server is refusing the request due to the entity of the request is in a format that is not supported by the method on the target resource.

  • 416 Request Range Not Satisfiable

    The client has asked for a portion of the fle, bu the server cannot supply that portion.

  • 417 Expectation Failed

    The server cannot honor the request; the server cannot meet the requirements of the Expect request-header field.

  • 425 Upgrade Required

    The client should upgrade and switch to a different protocol.

Server Error - 5xx

The value of 5xx is server error.It means that the server failed to accomplish the valid request.

  • 500 Internal Server Error

    Misconfiguration or the server's at fault; it is given when no more specific message is suitable.

  • 501 Not Implemented

    The method cannnot be handled. The server does not support the functionality required to fufill the request.

  • 502 Bad Gateway

    The gateway machine is having a problem. The gateway server received an invalid response from the upstream server it accessed in attempting to fulfill the request.

  • 503 Service Unavailable

    The server is under maintenance or it is tempporary overloaded.

  • 504 Gateway Timeout

    The gateway server did not receive a timely response from the upstream server.

  • 505 HTTP Version Not Supported

    The server does not support the HTTP version used in the request.
