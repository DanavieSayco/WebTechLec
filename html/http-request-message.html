HTTP Request Message

Request Line

The Request-Line begins with a method token/request method, followed by the Request-URI and the protocol version, and ending with CRLF (terminated line consisting of three spaced-separated items).

  • Request Methods

    It passes the information about the request and the client to the server

  • Request URI
  • HTTP Protocol Version

    A server supporting HTTP version 1.1 will return the version information. (e.g. HTTP-Version = HTTP/1.1)

Request Header Fields

Request Header Fields provide information about the request messages requested by the client.

Request Message

Example of a Request Message body:

GET /sample.html HTTP/1.1
User-Agent: Mozilla/4.0 (compatible; MSIE5.01; Windows NT)
Host: www.sample.com
Accept-Language: en-us
Accept-Encoding: gzip, deflate
Connection: Keep-Alive
